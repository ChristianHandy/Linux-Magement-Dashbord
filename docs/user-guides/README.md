## Contents

Each guide covers the following chapters:

1. **Getting Started** — System requirements, login, and UI overview
2. **Dashboard** — Customizable widget system with drag-and-drop
3. **Host Management** — Adding, editing, and tagging servers and PCs
4. **Update Manager** — Remote system updates and update history
5. **VM Controllers** — Proxmox and other hypervisor integrations
6. **Storage Controllers** — TrueNAS and storage system management
7. **Disk Management & SMART** — Disk health monitoring and SMART analysis
8. **CheckMK Integration** — Monitoring integration and token authentication
9. **User Management** — RBAC roles, user accounts, and permissions
10. **Plugins & Extensions** — Addon system and custom plugin development
